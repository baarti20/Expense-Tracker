# Expense Tracker

A simple and intuitive web application to help you manage your personal finances. Track your income and expenses with ease, visualize your spending habits, and take control of your money.

## Features

- **Transaction Management:** Add, edit, and delete income and expense transactions.
- **Financial Summary:** View your total balance, total income, and total expenses at a glance.
- **Transaction History:** See a detailed list of all your past transactions.
- **Search & Filter:** Easily search for specific transactions and filter them by category, type (income/expense), or date range.
- **Data Visualization:** A dynamic bar chart breaks down your income and expenses by category.
- **Data Portability:** Export your transaction data to a CSV file or import data from a CSV.
- **Dark Mode:** Switch between light and dark themes for comfortable viewing.
- **Local Storage:** Your data is saved securely in your browser's local storage.
- **Responsive Design:** Use the app on any device - desktop, tablet, or mobile.

## Technologies Used

- **HTML5:** For the structure and content of the application.
- **CSS3:** For styling, including a responsive layout and a dark mode theme.
- **JavaScript (ES6+):** For all the application logic and interactivity.

## Getting Started

To run this project locally, simply follow these steps:

1.  **Clone the repository (or download the files):**
    ```bash
    git clone https://github.com/baarti20/Expense-Tracker
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd expense-tracker
    ```
3.  **Open the `index.html` file in your web browser.**

That's it! The application will be running in your browser.

## How to Use

1.  **Add a Transaction:**
    - Fill in the "Add New Transaction" form with the date, title, amount, and category.
    - Select whether it's an "Income" or "Expense".
    - Click "Add Transaction".

2.  **View & Filter Transactions:**
    - Your transactions will appear in the "Transactions" list.
